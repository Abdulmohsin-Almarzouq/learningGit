1) git init: initialize project to use git
2) git add .: adds all changes to be saved
3) git add *filename*: add single file to be saved
4) git commit -m 'message': saves changes with message
5) git push origin master: push changes to github master
6) git push origin newBranch: push changes to github newBranch
7) git checkout -b newBranch: create a new branch  called newBranch
8) git status: check status of changes
9) git log: see all previos saved changes
10) git checkout *commit hash*: travel back to old commit
